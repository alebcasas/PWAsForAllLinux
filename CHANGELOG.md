# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of PWAsForAllLinux
- GTK4-based graphical user interface
- WebKitGTK web engine integration
- Install any website as a Progressive Web App
- Automatic manifest detection from URLs
- Isolated profiles for each PWA
- System desktop integration (.desktop files)
- Custom icons for each PWA
- Configurable window dimensions
- Three display modes: standalone, minimal-ui, fullscreen
- Settings panel with customization options
- Hardware acceleration support
- Universal installer script for all Linux distributions
- Support for x86_64 and ARM64 architectures
- Compatible with GNOME, KDE, XFCE, MATE, Cinnamon, LXQt, Budgie, i3/Sway

### Package Formats
- DEB package for Debian/Ubuntu/Linux Mint
- RPM package for Fedora/openSUSE/RHEL
- AppImage for universal compatibility

### Documentation
- Comprehensive README with step-by-step instructions
- Installation guide for each major distribution
- FAQ section
- Contributing guidelines
