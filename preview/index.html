✨ Características Principales

🌐 Compatible con Todas las Distribuciones

Ubuntu, Fedora, Arch Linux, openSUSE, Debian, Linux Mint, Manjaro y más. El instalador detecta automáticamente tu distribución.

🖥️ Todos los Entornos de Escritorio

GNOME, KDE Plasma, XFCE, MATE, Cinnamon, LXQt, Budgie, i3/Sway. GTK4 funciona en todos.

🔒 Perfiles Aislados

Cada PWA tiene su propio perfil independiente. Cookies, caché y datos separados para mayor privacidad.

🚀 Integración Nativa

Iconos en el menú de aplicaciones, entradas .desktop, integración completa con el sistema.

⚙️ Configuración Flexible

Tamaño de ventana personalizable, modos de visualización, aceleración por hardware, temas.

📦 Múltiples Formatos de Paquete

DEB para Debian/Ubuntu, RPM para Fedora/openSUSE, AppImage universal, instalador automático.

📥 Instrucciones de Instalación

Método Rápido (Recomendado)

# Descargar el instalador curl -fsSL https://raw.githubusercontent.com/pwasforalllinux/pwasforalllinux/main/scripts/install.sh -o install.sh # Dar permisos y ejecutar chmod +x install.sh sudo ./install.sh

Compilar desde Código Fuente

# 1. Clonar el repositorio git clone https://github.com/pwasforalllinux/pwasforalllinux.git cd pwasforalllinux # 2. Instalar Rust (si no está instalado) curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh # 3. Instalar dependencias (Ubuntu/Debian) sudo apt install build-essential libgtk-4-dev libwebkit2gtk-4.1-dev # 4. Compilar cargo build --release # 5. Ejecutar ./target/release/pwasforalllinux

Dependencias por Distribución

Ubuntu / Debian

sudo apt install libgtk-4-1 libwebkit2gtk-4.1-0

Fedora

sudo dnf install gtk4 webkit2gtk4.1

Arch Linux

sudo pacman -S gtk4 webkit2gtk-4.1

openSUSE

sudo zypper install gtk4 libwebkit2gtk3

🎮 Cómo Usar

Paso 1: Abrir la Aplicación

Busca "PWAsForAllLinux" en el menú de aplicaciones de tu sistema o ejecuta pwasforalllinux en la terminal.

Paso 2: Añadir una PWA

Haz clic en "Add PWA", ingresa la URL del sitio web (ej: https://web.whatsapp.com) y haz clic en "Auto-detect from URL".

Paso 3: Instalar

Haz clic en "Install PWA". La aplicación aparecerá automáticamente en tu menú de aplicaciones.

Paso 4: Ejecutar

Abre la PWA desde el menú de aplicaciones o haz clic en "Launch" dentro de PWAsForAllLinux.
